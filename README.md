# Mesin Slot Tech Stack

Aplikasi web sederhana untuk mengacak kombinasi teknologi (Tech Stack) menggunakan simulasi mesin slot.

## Fitur

- Mengacak 4 kategori teknologi: Runtime, Backend, Frontend, dan Database.
- Animasi putaran slot yang halus.
- Tampilan hasil dalam popup modal.

## Cara Menjalankan

1. Clone atau download repository ini.
2. Buka file `index.html` langsung menggunakan web browser (Chrome, Firefox, Edge, dll).
3. Klik tombol "SPIN!" untuk mulai mengacak.

## Teknologi

- HTML5
- JavaScript (Vanilla)
- Tailwind CSS (via CDN)
